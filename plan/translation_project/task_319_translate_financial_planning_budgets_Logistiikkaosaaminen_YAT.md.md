# Logistiikkaosaaminen

Logistiikka on yrityksen elinehto. Se on tieto-, materiaali- ja rahavirtojen hallintaa koko toimitusketjussa.

## 1. Logistiikan määritelmä
Logistiikka tarkoittaa:
* **Materiaalivirta:** Tuleva, sisäinen ja lähtevä tavaravirta raaka-aineista loppukuluttajalle.
* **Informaatiovirta:** Tietojen kulku tilauksista, varastotasoista ja aikatauluista.
* **Rahavirta:** Maksujen liikkuminen ketjussa.

## 2. Keskeiset periaatteet (JOT & FIFO)
* **JOT (Just On Time):** Oikeat tuotteet, oikeaan aikaan, oikeassa paikassa ja oikealla laadulla minimikustannuksin.
* **FIFO (First In, First Out):** Ensimmäisenä varastoon tullut tavara lähtee sieltä ensimmäisenä (tärkeää erityisesti pilaantuvien tuotteiden kohdalla).
* **Hukan minimointi:** Turhien vaiheiden ja varastoinnin poistaminen.

## 3. Logistiikan tavoitteet
Tehokkaalla logistiikalla tavoitellaan:
1. **Parempi asiakaspalvelu** -> Lisää tuloja.
2. **Alhaisemmat kustannukset** -> Parempi voitto.
3. **Nopeampi kierronopeus** -> Vähemmän sitoutunutta pääomaa.
4. **Korkea toimitusvarmuus** -> Asiakastyytyväisyys.

## 4. Kilpailukyvyn osatekijät
* **Laatu:** Asiakas saa sen mitä haluaa, silloin kun haluaa.
* **Hinta:** Kilpailukykyinen hinta-laatusuhde.
* **Toimitusnopeus ja -varmuus:** Luotettava ja nopea toimitus.

## 5. Logistiset tunnusluvut
* **Varaston kiertonopeus:** Ostot hankintahintaan / keskivarasto.
* **Keskimääräinen varastossaoloaika:** 365 / varaston kiertonopeus.
* **Palveluasteprosentti:** (Toimitetut tilaukset / Tilaukset yhteensä) * 100.
* **Toimitusvarmuus:** (Virheettömät tilaukset / Toimitusten määrä) * 100.

## 6. Sopimukset ja toimituslausekkeet
Kauppasopimuksissa sovitaan:
* Tavaran erittely ja hinta.
* Maksuehdot.
* **Toimituslausekkeet (esim. Incoterms):** Määrittelevät kuka vastaa kustannuksista ja milloin vaaranvastuu siirtyy myyjältä ostajalle.

---
**Huomautus:** Huolehdi aina, että myyntisaatavien kiertonopeus on nopeampi kuin ostovelkojen kiertoaika!
