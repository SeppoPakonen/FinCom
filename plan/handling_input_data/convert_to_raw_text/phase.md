# Phase: Convert to Raw Text

The goal of this phase is to convert all non-text files in the `unprocessed/` directory into raw text files. This includes PDFs, Word documents, Excel spreadsheets, PowerPoint presentations, and HTML files.

## Tasks
- Extraction of text from PDF files using appropriate tools.
- Extraction of text from Word (.doc, .docx, .odt) files.
- Extraction of data/text from Excel (.xlsx) files.
- Extraction of text from PowerPoint (.pptx, .odp) files.
- Extraction of main content from HTML files (ignoring supporting assets).
