# Glossary: Digital Financial Statements

This glossary explains key concepts related to digital financial statement reporting (XBRL/IXBRL) and sustainability reporting.

| Term | Explanation |
| :--- | :--- |
| **ESEF** | *European Single Electronic Format*. Common reporting format for European listed companies. |
| **ESRS** | XBRL taxonomy according to the EU sustainability reporting directive. |
| **IXBRL** | *Inline eXtensible Business Reporting Language*. Combines machine-readable data and human-readable XHTML page. |
| **PIE entity** | *Public Interest Entity*. Entity significant from a public interest perspective (e.g. listed companies, banks, insurance companies). |
| **PRH identifier** | Term used in accounting legislation for IXBRL taxonomy elements. |
| **Taxonomy** | Structured information model that defines how information is presented electronically. |
| **Tagging** | Marking financial statement items with taxonomy-compliant codes (elements). |
| **XBRL** | Machine-processing oriented reporting language for transferring financial data. |
| **XHTML** | Web page markup language used as the basis for IXBRL reports. |
