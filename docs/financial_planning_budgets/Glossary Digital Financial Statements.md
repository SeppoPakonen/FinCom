# Glossary: Digital Financial Statements

This glossary explains key concepts related to digital financial reporting (XBRL/IXBRL) and sustainability reporting.

| Term | Explanation |
| :--- | :--- |
| **ESEF** | *European Single Electronic Format*. Uniform reporting format for European stock exchange companies. |
| **ESRS** | XBRL taxonomy according to the EU's sustainability reporting directive. |
| **IXBRL** | *Inline eXtensible Business Reporting Language*. Combines machine-readable data and human-readable XHTML page. |
| **PIE entity** | *Public Interest Entity*. Entity significant from the public interest perspective (e.g., stock exchange companies, banks, insurance companies). |
| **PRH identifier** | Term used in accounting laws for IXBRL taxonomy elements. |
| **Taxonomy** | Structured information model that defines how information is presented electronically. |
| **Tagging** | Marking financial statement items with taxonomy-compliant codes (elements). |
| **XBRL** | Reporting language designed for machine processing of financial data. |
| **XHTML** | Markup language for web pages, used as the basis for IXBRL reports. |
