# Exercise: Document Formatting (Word/Writer)

This assignment guides you in creating and formatting a professional document according to the given instructions.

## Steps of the Assignment

1. **Open the template text:** Use the file `Computer driving license assignment text.md`.
2. **Clean the text:** Remove unnecessary line breaks and blank lines under headings.
3. **Basic formatting:**
    * Font: **Arial**
    * Size: **12 pt**
    * Line spacing: **1.5**
    * Alignment: **Both margins** (Justified)
    * Hyphenation: **On**
4. **Headings:** Replace section numbers (1, 2, 3) with descriptive subheadings:
    * **Objective**
    * **Result**
    * **Tool**
5. **Indentation:** Use dependent indentation.
    * Subheadings start from the margin (C0).
    * Body text starts at **4.6 cm** (C2).
6. **Headers and footers:**
    * **Header:** Left side appropriate image (width 1/4 of page), right side fixed date.
    * **Footer:** Left side your name, right side page numbering in the format `page x (y)`.
7. **Final touches:** Use preview to ensure paragraphs don't break untidily across pages.
8. **Save:** Save as `Computer Driving License Qualification.rtf` or `.docx`.
