# Exercise: Formatting a Long Document

In this task, you practice using automatic functions in managing a long document.

## Task Steps

1. **Open the base text:** Use the file `Base text: Digital Photography.md`.
2. **Headings:** 
    * Use the **Heading 1** style for main headings.
    * Use the **Heading 2** style for subheadings.
3. **Cover page:** 
    * Add a blank page at the beginning (*Ctrl + Enter*).
    * Create a cover page with the title of the work, your name, and the date.
4. **Table of Contents:**
    * Add an automatic table of contents on its own page after the cover page.
5. **Page numbering:**
    * Add page numbers to the footer on the right.
    * Numbering may only start after the table of contents. (Tip: Use a section break).
6. **Images:**
    * Add at least two images related to the topic in the text.
    * Add automatic captions to the images.
7. **Save:** Save as `Long_document_photography.docx`.

---
**Goal:** Create a professional and readable whole in which automatic lists work correctly.
