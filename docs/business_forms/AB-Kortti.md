# Computer User's AB Card (Review)

This is a summary of the Computer User's AB Card qualification and its key competency areas. The AB Card is intended for advanced computer users.

## 1. Structure of the Qualification
* **Prerequisite:** Computer User's A Card.
* **Content:** 1 compulsory module + 3 optional modules.
* **Demonstration exam:** Duration 2 h 15 min.

## 2. Software Integration (Compulsory)
* **Objective:** Using input data in different programs.
* **Compatibility:** Different operating systems and software versions.
* **Techniques:** Linking, embedding, import and export.
* **Management:** Customizing user interfaces, installations, backups and information security.
* **Teamwork:** Email management, calendars and task delegation.

## 3. Optional Modules (Examples)

### Text Processing
* Styles, document standards and automation.
* Mailing lists and forms.
* Using macros in routines.

### Spreadsheet Calculation
* Complex functions (IF, LOOKUP, SUMIF).
* Data analysis (Cross-tabulation, goal seek).
* Linking charts and matrix calculation.

### Databases
* Designing relational databases (primary and foreign keys).
* Basics of SQL language.
* Custom reports and forms.

### Other Modules
* **Publishing and Image Processing:** Print production and PDF workflow.
* **Web Publishing:** Updating WWW pages and FTP.
* **Digital Communication:** Multimedia elements and interactivity.
* **Telecommunications:** Network structures and mobile technologies.
