# Computer User's AB Card: Contents and Objectives of the Qualification

This document defines the objectives and key contents of the modules that comprise the AB card module, which is part of the ICT professional qualification.

---

## 1. Overview
The qualification provides skills for effective use of information technology in working life. It is suitable for continuing education for the working population after completing the A-card.
- **Structure:** 1 compulsory and 3 optional modules.
- **Demonstration exam:** 2 times 15 min.

---

## 2. Sharing Computer Programs and Data (Compulsory)

### Objectives
- Versatile utilization of input data in different programs.
- Management of file formats, data transfer methods (linking, embedding) and group programs (email, calendar).
- Workstation maintenance, protection and backups.

### Key content
- **Operating system:** Customization, network stations, installations, print queues.
- **Data transfer:** Linking, embedding, importing/exporting, ready interfaces.
- **Office tools:** Image archives, equation editors, graphics.
- **Group programs:** Email settings, archiving, real-time communication, time management (calendars, task lists).

---

## 3. Text Processing (Optional)

### Objectives
- Creating uniform company document templates (bases) and forms.
- Managing multi-page and long documents (table of contents, indexes, styles).

### Key content
- **Formatting:** Styles, document standards, column layout, division.
- **Automation:** Merge functions (mass mailings, labels), macros, tables.

---

## 4. Spreadsheet Calculation (Optional)

### Objectives
- Creating calculation models, analysis and visualization of results.
- Using advanced functions and macros.

### Key content
- **Charts:** Printing, editing and linking to other programs.
- **Functions:** Min, Max, Average, If, Sumif, Countif, CountA, Lookup, Date and Financial functions.
- **Analysis:** Goal seek, filtering, cross-tabulation, matrix calculation, conditional formatting.

---

## 5. Databases (Optional)

### Objectives
- Designing, creating relational databases and ensuring data accuracy.
- Implementation of customized forms, reports and queries.

### Key content
- **Structure:** Multi-table relations, primary and foreign keys, referential integrity.
- **Queries:** AND/OR logic, search, update, delete and parameter queries. Utilizing SQL language.
- **Automation:** Macros, program code, dropdown menus.

---

## 6. Publishing and Image Processing (Optional)

### Objectives
- Producing print materials (combining text and images).
- Editing and optimizing images for different publishing channels.

### Key content
- **Theory:** Printing process, file formats (PDF), copyright, scanning.
- **Practice:** Sheet definition, text flow, drawing tools, color management, filters.

---

## 7. Web Publishing Management (Optional)

### Objectives
- Updating, building web pages and searching for information on the web.
- Efficient use of FTP services and search bots.

### Key content
- **Production:** Editing HTML code, using editors, optimizing images for web.
- **Communication:** Email and discussion groups in problem solving.

---

## 8. Digital Communication (Optional)

### Objectives
- Understanding multimedia possibilities and compiling digital presentations.
- Managing media elements (audio, video, animation).

### Key content
- **Elements:** GIF animations, digital video, audio file formats and transfer.
- **Process:** Content layering and interactivity.

---

## 9. Telecommunications (Optional)

### Objectives
- Managing telecommunications terminology, telephone networks and wireless data transfer.
- Understanding the flow of data and voice in different networks.

### Key content
- **Basics:** Information theory and basic technologies.
- **Networks:** Structure and services of telephone and data transmission networks.
- **Mobile:** Radio systems, basic structure of mobile networks and data transmission.
