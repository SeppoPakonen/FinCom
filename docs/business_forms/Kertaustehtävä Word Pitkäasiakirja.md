# Practice Task: Word Long Document

This task guides you in managing Word documents, using sections, and creating automatic lists.

---

## Steps of the Task

1. **Heading numbering:** Open the file "PRACTICE_TASK" and first create heading numbering for the document.
2. **Preparing the table of contents:** Add a forced **section break** before the first heading for the table of contents that will appear on its own page.
3. **Creating the table of contents:** Create a table of contents in the document using a ready-made template (e.g. "Official").
4. **Creating a cover page:**
    - Add another section break for the cover page.
    - Write in large font and using bold formatting on the cover page: **AB REVIEW**.
    - Center the title on the page.
    - Add a picture related to the topic above the title.
5. **Section management:** The document should now have 3 separate sections:
    - Section 1: Cover page
    - Section 2: Table of contents
    - Section 3: Text portion
6. **Removing header and footer links:**
    - Go to section 2 and remove the link to the previous section (Link to Previous) from both the header and footer.
    - Repeat the same for section 3.
7. **Filling in headers and footers:**
    - **Section 2 (Table of Contents):** Add the text *Table of Contents* to the center of the header.
    - **Section 3 (Text portion):**
        - Add the text *Computer AB-card* to the center of the header.
        - Add page numbering to the center of the footer.
        - Modify the page numbering to start from number 1.
8. **Finalizing:** Update the table of contents and save the completed document as **AB-card**.
