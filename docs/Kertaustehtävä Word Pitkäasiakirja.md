# Exercise: Word Long Document

This exercise guides you in managing Word documents, using sections and creating automatic lists.

---

## Steps of the Exercise

1. **Heading numbering:** Open the file "EXERCISE" and start by numbering the document's headings.
2. **Preparing the table of contents:** Add a forced **section break** before the first heading for the table of contents that will appear on its own page.
3. **Creating the table of contents:** Create a table of contents in the document using a ready-made template (e.g. "Official").
4. **Creating a cover page:**
    - Add another section break for the cover page.
    - Write in large font and using bold formatting on the cover page: **REVIEWING THE AB CARD**.
    - Center the title on the page.
    - Add an image related to the topic above the title.
5. **Section management:** The document should now contain 3 separate sections:
    - Section 1: Cover page
    - Section 2: Table of contents
    - Section 3: Text section
6. **Removing header and footer links:**
    - Go to section 2 and remove the link to the previous section (Link to Previous) from both header and footer.
    - Repeat for section 3.
7. **Filling in headers and footers:**
    - **Section 2 (Table of contents):** Add to the center of the header: *Table of Contents*.
    - **Section 3 (Text section):**
        - Add to the center of the header: *Computer User's AB Card*.
        - Add page numbering to the center of the footer.
        - Modify the page numbering to start from 1.
8. **Finalizing:** Update the table of contents and save the completed document as **AB-card**.
