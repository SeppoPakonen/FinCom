# Generated Documentation and Diagrams

This directory contains automatically generated documentation and diagrams from the exported data processing system.

## Diagrams

### 1. Data Structures Class Diagram
- **File**: `data_structures_class_diagram.puml` / `data_structures_class_diagram.png`
- **Description**: Shows the class structure of the data elements used in the system (Keywords, ECS elements, Constraints, Metadata, etc.)

### 2. Data Extraction Sequence Diagram  
- **File**: `data_extraction_sequence_diagram.puml` / `data_extraction_sequence_diagram.png`
- **Description**: Illustrates the sequence of operations when extracting data from documentation files

### 3. Network Relationships Diagram
- **File**: `network_relationships_diagram.puml` / `network_relationships_diagram.png`
- **Description**: Shows how different data elements are connected across documentation files

## Documentation Files

### 1. Network Verification
- **File**: `network_verification.md`
- **Description**: Verifies the networking capabilities between different data elements extracted from documentation files

### 2. Example Usage
- **File**: `example_usage.md`
- **Description**: Demonstrates how to use the data export and networking system in the Finnish Business Operations Repository

## Generation Process

These files were automatically generated by the `generate_documentation.py` script, which:
1. Analyzes the exported data from documentation files
2. Creates relationship graphs connecting different data elements
3. Generates PlantUML diagrams showing system architecture and relationships
4. Creates markdown documentation explaining the networking functionality
