# Computer User AB Card (Review)

This is a compilation of the Computer User AB Card qualification's content and key competence areas. The AB Card is intended for efficient computer users.

## 1. Qualification Structure
* **Prerequisite:** Computer User A Card.
* **Content:** 1 mandatory module + 3 elective modules.
* **Demonstration Exam:** Duration 2 h 15 min.

## 2. Program Interoperability (Mandatory)
* **Objective:** Using data entered once in different programs.
* **Compatibility:** Different operating systems and program versions.
* **Techniques:** Linking, embedding, import/export.
* **Management:** User interface customization, installations, backups, and security.
* **Teamwork:** Email management, calendars, and task delegation.

## 3. Elective Modules (examples)

### Text Processing
* Styles, document standards, and automation.
* Mass mailings and forms.
* Macro utilization in routines.

### Spreadsheet Calculation
* Complex functions (If, Lookup, SumIf).
* Data analysis (Cross-tabulation, goal seek).
* Chart linking and matrix calculations.

### Databases
* Design of relational databases (primary and foreign keys).
* Basics of SQL language.
* Custom reports and forms.

### Other Modules
* **Publishing and Image Processing:** Layout of print materials and PDF workflow.
* **Web Publishing:** Updating WWW pages and FTP.
* **Digital Communication:** Multimedia elements and interactivity.
* **Telecommunications:** Network structures and mobile technologies.
