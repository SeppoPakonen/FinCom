# Exercise: Formatting a Long Document

In this assignment, you will practice using automatic features for managing long documents.

## Steps of the Assignment

1. **Open the template text:** Use the file `Template Text: Digital Photography.md`.
2. **Heading formatting:**
    * Use the style **Heading 1** for main headings.
    * Use the style **Heading 2** for subheadings.
3. **Cover page:**
    * Add a blank page at the beginning (*Ctrl + Enter*).
    * Create a cover page with the work title, your name and date.
4. **Table of contents:**
    * Add an automatic table of contents after the cover page on its own page.
5. **Page numbering:**
    * Add page numbers to the footer on the right.
    * Numbering should only start from the page after the table of contents. (Tip: Use a section break).
6. **Images:**
    * Add at least two images related to the topic in the text.
    * Add automatic captions to the images.
7. **Save:** Save as `Long_document_photography.docx`.

---
**Goal:** Create a professional and readable document where automatic lists work correctly.
