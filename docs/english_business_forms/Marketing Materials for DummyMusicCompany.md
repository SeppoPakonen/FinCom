## Instructions for Visual Material Design

### Logo Implementation Guidelines
- Primary logo should work in both horizontal and stacked formats
- Include a simplified icon version for social media profiles
- Ensure readability at small sizes (e.g., business card)
- Create black-and-white versions for single-color printing

### Color Palette Specifications
- Primary Colors: Deep Purple (#4B0082), Gold (#FFD700)
- Secondary Colors: Navy Blue (#001F54), Silver (#C0C0C0)
- Accent Colors: Teal (#008080), Coral (#FF7F50)

### Typography Recommendations
- Headlines: Modern sans-serif (e.g., Montserrat or Roboto)
- Body text: Clean, readable font (e.g., Open Sans or Lato)
- Accent text: Script or decorative font for special elements only

### Business Card Design Concepts
- Front: Logo, company name, slogan, contact information
- Back: QR code linking to portfolio, space for handwritten personal note
- Finish: Matte with spot UV on logo for premium feel
