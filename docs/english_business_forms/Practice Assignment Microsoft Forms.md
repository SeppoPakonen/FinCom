# Practice Assignment: Microsoft Forms

The objective of this assignment is to learn how to build and manage digital survey forms using the Microsoft Forms tool.

## Assignment Steps

### 1. Creating the Form
* Log in to the Microsoft 365 service and open **Forms**.
* Select **New form** (do not use a ready-made template).
* Give the form a descriptive name (e.g., *Customer Satisfaction Survey* or *Registration Form*).

### 2. Questions and Types
* Create at least **10 questions**.
* Use at least **four different question types**:
    1. Choice
    2. Text
    3. Rating
    4. Date or Likert scale.

### 3. Appearance and Settings
* Customize the survey's **style** (theme, colors, or background image).
* Customize the **thank you message** that appears to the respondent after submission.
* Adjust settings so that you receive an **email notification** for each new response.

### 4. Sharing and Analysis
* Send the survey to at least two people.
* View the received responses using the *Responses* tab in the Forms service.
