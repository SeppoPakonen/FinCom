# Examples Directory

This directory contains shell scripts that demonstrate how to use the Python implementations of financial planning tools.

## Available Scripts

### `run_financial_demos.sh`
Runs all financial planning tools in sequence to demonstrate their basic functionality.

Usage:
```bash
./run_financial_demos.sh
```

### `run_individual_demos.sh`
Runs each tool individually with explanations of what each tool does.

Usage:
```bash
./run_individual_demos.sh
```

### `run_business_workflow.sh`
Demonstrates a complete business planning workflow using multiple tools together.

Usage:
```bash
./run_business_workflow.sh
```

### `how_to_customize.sh`
Shows how to customize the tools for specific business needs.

Usage:
```bash
./how_to_customize.sh
```

## Prerequisites

Before running these scripts, ensure you have:
- Python 3.x installed
- Required Python packages (pandas, numpy, etc.)

## About the Tools

These scripts demonstrate the following financial planning tools:
- YT4: Financial plans for Lapua and BusinessOulu
- YT5: Enhanced financial plan (Version 5)
- YT6: Startup profit planning
- YT22: Operating business profit planning
- YT23: Cash budget management
- YT24: Company valuation calculator
- YT25: Marketing budget and annual calendar

Each tool generates JSON output files with detailed financial projections and analyses.
